# logtape v0.1 Spec

## Event Envelope
Every line in JSONL is one event object.

Required top-level fields:
- `v` (string): must be `"0.1"`
- `ts` (string): RFC3339 with nanoseconds
- `kind` (string)
- `name` (string)
- `trace_id` (string, 32 hex)
- `span_id` (string, 16 hex)
- `attrs` (object)

Optional:
- `level` (string)
- `service` (string)
- `service_version` (string)

## HTTP Event (`kind="http"`, `name="http.server"`)
Minimum attrs:
- `http.method` (string)
- `url.path` (string)
- `server.duration_ms` (number)
- `http.response.status_code` (integer) when no `error.*`

Common attrs:
- `url.scheme`
- `url.query` (redacted if emitted)
- `network.protocol.version`
- `http.request.headers`
- `http.response.headers`

Body attrs (request and response):
- `http.*.body.size`
- `http.*.body.truncated`
- `http.*.body.sha256`
- `http.*.body.preview`
- `http.*.body.encoding` (`utf-8` or `base64`)

Error attrs:
- `error.type`
- `error.message`

## Redaction
- Header names matched case-insensitively via `--redact-headers`
- Query keys matched case-insensitively via `--redact-query`
- Redacted value is `[REDACTED]`

## Trace Context
- If inbound `traceparent` exists and valid: keep inbound trace_id, generate new span_id
- Otherwise: generate trace_id + span_id
- Inject upstream: `traceparent: 00-<trace_id>-<span_id>-01`
